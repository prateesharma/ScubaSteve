# scuba_steve
ROS package for the ScubaSteve AUV.

The following is the directory structure of the ROS package:
```
scuba_steve/
├── CMakeLists.txt
├── package.xml
├── README.md
├── setup.py
├── src/
    └── scuba_steve/
        ├── __init__.py
        ├── launch/
        ├── sim/
        ├── state_machine/
            ├── __init__.py
            ├── nodes/
            ├── state_machine/
            ├── states/
            └── utils/
        └── transducers/
└── test/
    └── scuba_steve/
        └── state_machine/
```
